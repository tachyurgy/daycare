7Violations & Inspection Data

Access mode

Monthly PDF   Predictable URLs

DHHS publishes monthly negative-action PDFs around the 5th of each month at {MM}-{YY}ccnegactions.pdf. A 10-year cumulative archive is also maintained.

Action types captured

  • Provisional license, probation
  • Denial, emergency suspension, suspension, revocation
  • Voluntary surrender in lieu of discipline
  • Fine, consent order, stipulated agreement

Key fields per action

  • Licensee / facility name, license number, license type
  • Action type + effective date
  • Grounds (tied to §71-1919 / Title 391 NAC)
  • County, city (address where applicable)

Automation strategy

  1. Scheduled curl on the 6th of each month to https://dhhs.ne.gov/licensure/Documents/{MM}-{YY}ccnegactions.pdf.
  2. Parse via pdftotext -layout + regex on license-# column.
  3. Diff against prior month; push new enforcement actions into ComplianceKit sales-trigger pipeline.
  4. Re-run the weekly roster parser documented in nebraska/SOURCES.md every Friday to track new entrants / closures.
  5. Supplement per-licensee detail via the HHS License Search CGI (no captcha).

FOIA path

Nebraska Public Records Statute (Neb. Rev. Stat. §84-712 et seq.). Most licensure data is already public; formal request only needed for structured CSV or pre-2016 records. Contact: DHHS.LicensureUnit@nebraska.gov.

8Outreach Signals

  • Predictable enforcement feed — monthly PDFs at known URL = automated "recently-cited" alert for sales.
  • No bulk email — enrichment required (Hunter/Apollo); phone coverage is strong.
  • Twice-a-year inspection cadence for centers ≥ 30 children creates a natural twice-yearly renewal anchor for ComplianceKit pitches.
  • Provisional-license transition (90-day cliff after first year) is a common failure point — offer ComplianceKit as "graduation insurance."
  • Step Up to Quality (NE QRIS) participants are premium prospects (voluntarily opting into more paperwork = receptive to compliance software).
  • Annual Immunization Report mandate is a recurring pain-point that ComplianceKit's Child File Management feature solves directly.
