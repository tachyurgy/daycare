6. Violations & Inspection Data

Primary sources

  • DCFS Sunshine: sunshine.dcfs.illinois.gov — dedicated transparency portal (data since 2010). Contains: Provider Lookup, Monitoring Reports, Investigation Actions, Annual Child Care Report.
  • Monitoring Reports: per-facility monitoring detail
  • Investigation Actions tracker: enforcement + CAN/ENR/COR action log

Data format

ASP.NET WebForms (v 2.3.2.18623) with __VIEWSTATE / __EVENTVALIDATION preservation required for postbacks. CSV export confirmed on Provider Lookup and Investigation Actions pages (Monitoring Reports is HTML-only). Severity coding: CAN (abuse/neglect), ENR (referred for revocation), COR (corrected), MON (monitoring), NFA (no findings). Propagation lag 48–72 hours after staff enters the action.

Freshness

  • 48–72 hours after DCFS action entry
  • Annual Child Care Report posted each Q2

Scraping / access strategy

  • Provider Lookup: Export button already validated (produced 8,586-row illinois_leads.csv).
  • Investigation Actions: same postback flow → CSV of all enforcement actions. This is the richest hot-leads source.
  • Monitoring Reports: per-facility only; iterate License #s; rate-limit 1 req/sec (has tripped IP blocks).
  • Filter Investigation Actions for Action IN ('CAN','ENR') AND Date >= now - 90 days.

FOIA path

Illinois FOIA, 5 ILCS 140. Request via DCFS.FOIA@illinois.gov. 5-business-day response (extendable to 10). Useful for pre-2010 records, un-redacted complaint files, and bulk CSV of monitoring-report detail.

7. Outreach Signals

  • Investigation Action = CAN (abuse/neglect) or ENR (referred for revocation) in last 90 days (available via CSV export) — highest-intent signal.
  • Repeat violation flag on Monitoring Reports (available via per-facility scrape).
  • License status = "Pending renewal" (available in Provider Lookup export).
  • Radon test overdue — required every 3 years; status posted with license (requires Monitoring Report inspection).
  • License expiration within 90 days (available in Provider Lookup — 3-year term per Sec. 407.40).
