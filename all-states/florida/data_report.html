1. Licensing Authority

Florida Department of Children and Families (DCF), Office of Child Care Regulation. DCF Licensing Counselors conduct all inspections. Five counties (Hillsborough, Broward, Palm Beach, Pinellas, Sarasota) operate delegated local licensing agencies enforcing DCF standards plus county enhancements.

DCF address 2415 N. Monroe Street, Suite 400, Tallahassee, FL 32303-4190
Helpline (855) 776-2729
Public search CARES Public Search
DCF landing myflfamilies.com/services/licensing/child-care-licensure

2. Primary Regulatory Citation

Statute: Florida Statutes Chapter 402 (§ 402.301 - § 402.319).

Administrative Code: Florida Administrative Code Chapter 65C-22.

Handbook (incorporated by reference): Child Care Facility Handbook (October 2021) — legally binding via 65C-22.

Inspection form: CF-FSP 5316 Standards Classification Summary — the binary compliant/non-compliant form inspectors fill out during the visit (32 categories for facilities).

Violation classification: Class 1 (immediate risk) / Class 2 (serious, e.g., ratios) / Class 3 (less serious administrative).

6. Violations & Inspection Data

Florida's inspection cadence is the most aggressive of any major state — minimum 2 unannounced inspections per year per licensed facility. Per-facility public data is available through CARES; bulk structured extract requires scraping or a Sunshine Law (F.S. Ch. 119) request — which Florida grants uncommonly liberally.

CARES Public Search

Per-facility HTML profile + inspection history + attached PDF reports. Not API-accessible. Dynamic rendering.

CF-FSP 5316 (32 inspection categories)

Every facility inspection scored on 32 binary compliant/non-compliant items keyed to the Handbook.

Class 1 / 2 / 3 severity

Immediate risk / serious / less serious. Ratio violations are Class 2 — inspectors count heads on-the-spot.

Records-request path

Florida Sunshine Law (F.S. Ch. 119 + Art. I § 24 Fla. Const.) — one of the strongest in the US. DCF Office of Public Records; ~5-10 business days.

Full analysis: violations.md

7. Outreach Signals

  • 2+ unannounced inspections per year — highest routine cadence among launch states. Urgency for inspection-readiness tools is correspondingly high.
  • Annual license renewal — 12-month cycle; every provider is a hot prospect every year.
  • 40-hour DCF introductory training — strictest intro training among CA/TX/FL. Credential-tracking pain is acute; Staff File module fits perfectly.
  • Florida Child Care Staff Credential + Director Credential — formal credentialing required for lead teachers and directors. Credential-lifecycle tracking is a must-have.
  • Clearinghouse background checks — statewide Clearinghouse portal requires ongoing roster maintenance; ComplianceKit can automate audit evidence.
  • Dual DCF + VPK paperwork — many providers also operate VPK under DOE oversight. Dual reporting is a natural wedge for a consolidated compliance tool.
  • Five-MSA concentration: Miami/Fort Lauderdale, Orlando, Tampa/St. Pete, Jacksonville, Cape Coral/Fort Myers — geographic targeting is straightforward.
  • High parent scrutiny — FL parents routinely check CARES profiles; clean-compliance is a marketable differentiator providers will pay for.

8. Next-Step Data Acquisition

  1. Scrape CARES Public Search incrementally by zip code (FL has ~1,000 unique zips active for child care). Priority counties: Miami-Dade, Broward, Orange, Hillsborough, Duval, Palm Beach, Pinellas, Lee, Polk, Seminole.
  2. Join with `florida_leads.csv` — anchor by license number; fill missing fields (type, capacity, address).
  3. Download + OCR inspection PDFs — normalize to (license_number, visit_date, visit_type, class_1_2_3, handbook_section, narrative, corrective_action, status, sanctions).
  4. File Sunshine Law request to DCF for full database extract per phrasing in violations.md (inspection records, violations, enforcement actions, 2019-present, in CSV/Excel). FL's records regime is favorable.
  5. Supplementary requests to local licensing agencies in Hillsborough / Broward / Palm Beach / Pinellas / Sarasota for any county enhancements.
  6. Cross-reference Early Learning Coalition network (subsidy + VPK) for enrichment on provider types and federal participation.
  7. DCF child fatality dataset (5-year rolling) is a public-facing data source for high-severity narrative content and SEO authority.
