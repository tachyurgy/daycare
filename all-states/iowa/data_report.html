6. Violations & Inspection Data

Primary source

Iowa HHS Titan Compliance Report (live ASP.NET paginated app) exposes per-provider annual compliance evaluations + complaint reports + corrective actions. Search supports county / city / provider-type filters.

  • Per-facility detail: rule-citation itemization, compliance/non-compliance per item, corrective action + due date, final disposition.
  • Complaint side: intake date, redacted allegations, investigation outcome (substantiated / unsubstantiated / partial), sanctions.

Secondary source

Socrata catalog data.iowa.gov dataset 5kd5-9khx — currently link-only (empty columns metadata); re-check quarterly for row exposure. Federal harvest entry: data.gov (resource URL broken).

Format

ASP.NET WebForms VIEWSTATE No Cloudflare block Polite Playwright or requests.Session() scrape works at ≤2 req/sec.

Freshness

Titan reflects licensor visits ≤72 hrs. Iowa runs an annual unannounced on-site at every licensed center (IAC 441-109.3); license term is 2 years (provisional 6 mo).

Open-records path

Iowa Code Chapter 22 — request to Iowa HHS Records Custodian; typical 10–20 business days. Child-abuse registry details remain confidential under Iowa Code 235A. Report abuse/fraud: hhs.iowa.gov/report-abuse-fraud.
