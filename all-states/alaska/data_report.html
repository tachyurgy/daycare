6. Violations & Inspection Data

Anchorage is the outlier. MOA publishes a Socrata open dataset of child care inspections (2008-present) — the richest structured inspection dataset available among the five smallest states and one of only a handful in the country at the municipal level. Rest-of-state (via AKCCIS) is per-facility search only.

Anchorage (Socrata)

Dataset abmr-hyh6. CSV / JSON / API. Fields: facility, date, type, 14 inspection categories, AMC 16.55 citations, narrative, correction deadline, lat/lon via pqqp-cdt5.

Rest of state (AKCCIS)

Per-facility search. License status + compliance summary. No bulk export JS-rendered

Merge step

Deduplicate AKCCIS + MOA by license number + address. MOA dataset is tabular; AKCCIS requires headless-browser enumeration.

Records-request path

AS 40.25 Public Records Act — dual requests to CCPO (statewide) and MOA Health Dept (Anchorage).

Full analysis: violations.md

7. Outreach Signals

  • Biennial license cycle — 2-year cadence; half the market is in renewal at any time.
  • Dual-jurisdiction complexity — Anchorage providers operate under both 7 AAC 57 AND AMC 16.55. Double-compliance pain is a strong buying signal for multi-rule software.
  • Barrier Crimes Act — enumerated disqualifier list requires ongoing variance tracking; maps directly to Staff File module.
  • Serious incident 24-hour reporting (7 AAC 57.545) — native use-case for ComplianceKit's Incident Log feature.
  • SEED Registry integration opportunity — SEED is the canonical PD registry; a SEED integration would eliminate a major manual-entry pain for every center.
  • 5 primary markets cover ~90% of licensed facilities: Anchorage, Fairbanks, Mat-Su (Wasilla/Palmer), Juneau, Kenai Peninsula.
  • Distributed rural / bush — remaining ~10% are small, remote; lower ARPA but higher retention.

8. Next-Step Data Acquisition

  1. Pull Anchorage Socrata immediately: curl 'https://data.muni.org/resource/abmr-hyh6.json?$limit=50000'. Get tabular inspection history for ~50% of Alaska's market in one API call.
  2. Pull companion geo dataset pqqp-cdt5 for lat/lon enrichment.
  3. Scrape AKCCIS via Playwright: enumerate by community/zip across all Alaska census areas. Build rest-of-state registry.
  4. File AS 40.25 records requests — one to CCPO (full statewide AKCCIS extract, 7-year window), one to MOA (full facility roster including MOA-licensed sites not yet migrated post-Oct-2024).
  5. Enrich via SEED Registry (seedalaska.org) — staff-level PD completeness as additional outreach signal ("Your room X is 4 hours short of the annual floor").
  6. Expand leads to Kodiak, Homer, Sitka, Ketchikan, Bethel, Nome, and tribal/DoD facilities outside the five primary markets.
