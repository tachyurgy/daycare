6. Violations & Inspection Data

Primary source

DLBC Find-a-Facility + ccl.utah.gov expose per-facility current license status, 36 months of inspection reports, and confirmed findings of rule non-compliance. Per-inspection URL: https://ccl.utah.gov/ccl/public/checklist/{ID} — e.g. 632272, 622289.

Format

DLBC blocks bots (403) No Socrata No ArcGIS Server-rendered HTML at the CCL public-checklist host is accessible; enumerate integer IDs with Playwright.

Freshness

Annual announced inspection per facility + unannounced complaint visits. Public rolling window = 36 months. Older compliance history requires a GRAMA request.

Enforcement

DLBC uses "OL Guidelines for Corrective Actions and Civil Money Penalties" — graduated ladder: written notice → corrective action plan → CMP → conditional / denial / revocation.

GRAMA (open records)

Utah Code §§ 63G-2-101 et seq. — 10-business-day response window (extendable). BCI GRAMA info: bci.utah.gov/grama-requests. Utah State Auditor overview: auditor.utah.gov/news/grama.

Consumer-facing reference

KUTV "Check Your Health" — how to use DLBC compliance search (parent-facing).

7. Outreach Signals

  • Care About Childcare (CAC) Registry required for every licensed staff — ComplianceKit integration = direct efficiency pitch
  • CMP assessed in last 36 mo — signals sustained compliance pain; strong buyer
  • Mixed-age group operators (unique Utah "oldest-minus-one-age-band" formula) — rule complexity is mis-applied often; ComplianceKit ratio calculator is a direct solve
  • AQI / wildfire outdoor-play rule — enforced per-day; emergency ops module fit
  • Shaken-Baby / AHT within 30 days for infant staff — narrow tracking window, easy to miss
  • Silver State Stars QRIS equivalent (CAC Quality Improvement) — voluntary; participation = marketing-savvy buyer persona
