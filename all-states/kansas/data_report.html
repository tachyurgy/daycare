7Violations & Inspection Data

Access mode

Captcha-locked   Per-facility only

KDHE OIDS publishes noncompliance findings per facility. Only cited regulations appear (compliant regs are hidden). reCAPTCHA + ASP.NET ViewState prevents batch enumeration.

Reporting window

Last 3 years of inspection results retained on OIDS.

Typical inspection cadence: initial, annual, and unannounced monitoring/complaint follow-ups.

Key fields exposed per facility

  • Facility name, license #, program type (11 categories), owner
  • Address, city, county (all 105), ZIP
  • Inspection/survey date + type
  • Cited K.A.R. or K.S.A. regulation number + short description
  • Noncompliance narrative

Scraping strategy

  1. Seed with 770 facilities in kansas_leads.csv (or a larger KORA-sourced roster).
  2. Playwright + 2Captcha / anti-captcha API to solve reCAPTCHA v3 per lookup. Budget ~$1.50 at current $0.002/solve rate.
  3. Parse HTML results; normalize citations to K.A.R. 28-4-xxx numeric keys.
  4. Preferred sanctioned path: KORA request for bulk inspection data (skip captcha entirely).

FOIA / KORA path

Kansas Open Records Act (K.S.A. 45-215 et seq.). KDHE Child Care Data Request page is the official channel — kdhe.ks.gov/2185. Request "all inspection results and cited noncompliance records for licensed child care facilities, 2023-2026, in CSV/Excel." Statutory response 3 business days. Fees: copying + staff time, often waived for news/research.
