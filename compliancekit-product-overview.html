State Coverage

Built for Your State's Exact Requirements

Every state has different forms, different frequencies, different violation systems. ComplianceKit knows them all.

FL
Florida
  • Inspection Framework CF-FSP 5316 Standards Classification Summary—32 categories, hundreds of line items
  • Inspector Title DCF Licensing Counselor, minimum 2 unannounced visits per year
  • Violation System 4-class system. Class 1 (immediate danger): $100–$500/day. 3rd Class 1 = license revoked.
  • Public Record CARES database—every violation permanently searchable by any parent
  • Key Forms Tracked Form 680 immunizations, DH 3040, Level 2 screening, fire drill logs (12-month retention)
TX
Texas
  • Inspection Framework 3 evaluation forms: 7259 (Personnel), 7260 (Children), 7261 (Center Records)
  • Inspector Title HHSC Licensing Representative, annual unannounced + complaint-driven
  • Violation System Progressive enforcement: corrective action, probation, adverse action. Up to $1,000/violation.
  • Public Record Public Compliance History Search—all deficiencies visible to parents
  • Key Forms Tracked Form 1100 (daily), 2935, 2941, 7263 (drill log), 2948 (Plan of Operation), 20+ more
CA
California
  • Inspection Framework CARE Tools with 9 domains: supervision, records, food service, physical plant, ratios, and more
  • Inspector Title CCLD Licensing Program Analyst (LPA), 4 inspection types including random unannounced
  • Violation System Type A ($50–$150/day, compounding) and Type B. 2nd citation in 12mo = $150/violation.
  • Public Record CCLD Facility Search—inspection history searchable online
  • Key Forms Tracked LIC 700, 701, 702, 613A, 610 + LIC 9148 (earthquake), CDPH 286, 17+ per child file
