7Violations & Inspection Data

Access mode

Open HTML   Per-provider URL   No auth

idahochildcarecheck.org is the best per-facility violation/incident dataset among our assigned states. Published as rolling HTML with stable CSS selectors.

Inspection cadence

~1 unannounced inspection per year per facility, plus complaint-driven follow-ups. Critical violations require immediate correction or a scheduled follow-up visit.

Public data starts October 2017; minimum 3-year retention.

Key fields exposed per provider

  • Aggregate: total inspections, passes, fails ("13 Inspections Conducted: 10 passed 3 failed")
  • Per-inspection: date, pass/fail, type (Annual / Investigation / Follow-Up), resolution status
  • 31 numbered inspection criteria (supervision, CPR, ratios, food safety, emergency planning)
  • Inspector comments (narrative text)
  • Substantiated incidents: category, date, description, resolution

Scraping strategy

  1. Search-page enumeration: /search?page={N} for N in [0, 207]. 10 providers per page. Already captured for leads CSV.
  2. Per-provider scrape: iterate integer IDs extracted from search page; fetch /provider/{id}.
  3. Parse HTML with BeautifulSoup; target Drupal views-field-* selectors (stable).
  4. Weekly incremental sync: track latest-seen inspection date per provider; re-fetch only changed records.
  5. Conservative rate limit: 1-2 req/sec with exponential backoff on 5xx.

FOIA path

Idaho Public Records Act (Idaho Code §74-101 et seq.). IDHW is the custodian. Standard response window 3-10 business days. Delegated regional public health districts also subject to PRA. Use PRA to obtain: complete provider roster in CSV with addresses & emails; pre-Oct-2017 historical inspection records; full substantiated-incident narratives.

8Outreach Signals

  • Best-in-class public data — per-facility violation counts, substantiated incidents, and inspector comments all publicly accessible. Enables precise lead scoring by "risk score."
  • 0% email fill — emails not exposed on Idaho Child Care Check. Enrichment mandatory.
  • Point-based ratio is unique — providers often need calculator tooling; ComplianceKit can include an Idaho-specific 12-point ratio calculator.
  • HB 243 (2025) deregulation — ongoing changes to point values create confusion; time-sensitive sales angle.
  • Seven public health districts each conduct inspections with slightly different emphasis. ComplianceKit checklists can be tuned per district.
  • Family Daycare Homes (≤ 6 children) are exempt from state license; voluntary licensure unlocks ICCP subsidy — a subsegment actively adopting compliance tooling.
  • Boise's additional city license is stricter than state minimums — Boise operators are highest-intent prospects.
