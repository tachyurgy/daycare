6. Violations & Inspection Data

Primary source

Per-facility pages on the Child Care Locator expose every monitoring visit for the last 3 years, with per-rule compliant / non-compliant checklist items, substantiated complaint narratives, and corrective action plans.

  • Facility profile: https://childcarefind.okdhs.org/providers/{LICENSE_ID} — e.g. K830000229 (Grace Child Development Center, 4-Star)
  • Per-visit checklist: https://childcarefind.okdhs.org/licensing-history/{LICENSE_ID}/{YYYY-MM-DD} — e.g. K830025060 @ 2026-01-07
  • 35–36 items graded C / NC / NR, ratios by age, fire/health inspection dates, substantiated-complaint summary

Format

HTML only server-rendered No Socrata, no ArcGIS, no CSV roster. Scrapeable with BeautifulSoup; per-visit checklist may need Playwright for full fidelity.

Freshness

Updated within days of each unannounced monitoring visit. Minimum cadence 1×/yr; centers typically see 2–4 visits/yr. Public window = last 3 years; older records via in-person file inspection.

Related registries

  • Joshua's List — Restricted Registry (people barred from licensed child care; searchable web tool; restricted use per 10 O.S. § 405.3)
  • OKC-County Health Department — local food/sanitation inspections layered on top of OKDHS

FOIA / Open Records

Oklahoma Open Records Act (51 O.S. §§ 24A). Facility licensing files are explicitly open for inspection per OAC 340:110-1 Part 14; complaint narratives confidential except for de-identified summaries. Requests to OpenRecords@okdhs.org.

7. Outreach Signals

  • Reaching for the Stars QRIS rating (1★/2★/3★) visible on every Locator profile — unstarred centers are more outreach-sensitive (ratings tied to subsidy reimbursement)
  • Non-compliance count on recent visits — e.g. "30 of 35 Areas in compliance" signals CPR / training / documentation gaps ComplianceKit directly fixes
  • Substantiated complaint within 24 months = high-intent buyer (post-crisis remediation)
  • Subsidy contract number present = revenue-sensitive to compliance lapses (state pulls subsidy on unresolved NCs)
  • Tornado-drill log requirement March–October — climate-specific pain point unique to OK/KS/TX
