6. Violations & Inspection Data

Primary source

Aithent ALiS Licensee Search — per-facility profile exposes a Regulatory Actions tab with inspection reports, complaint investigation outcomes, enforcement actions. Supports public access via the PubliSearch=Y flag on the Protected path. Same backend serves findchildcare.nv.gov.

Format

ASP.NET VIEWSTATE No Cloudflare block No Socrata / ArcGIS Playwright is the simplest enumeration path.

Freshness

2 unannounced inspections per 12-month licensing period (NRS 432A.180 / NAC 432A.190) + complaint follow-ups. Aithent reflects filings within days.

Aggregate reports

Statewide Annual Licensing Report (latest Jan–Dec 2022): total licensed facilities by type, substantiated / unsubstantiated complaint counts, jurisdiction breakdown. A "Serious Injury, Death, and Substantiated Abuse & Neglect 2023–2025" report is referenced in agency copy but URL is not stable — request via DSS public-records desk.

County-local gaps

Clark County family-home licensing is separate from state CCL (county business-license layer). Washoe legacy records (pre-7/1/2024) live at county level. These records are not in Aithent — require local records request.

Open-records path

Nevada Public Records Act — NRS 239. DHHS portal: dhhs.nv.gov/about/publicrecordsrequest. DHS: dhs.nv.gov. Response window: 5 business days for acknowledgment.

7. Outreach Signals

  • 24-hour annual training (NRS 432A.1775) — higher than most states; sustained tracking burden = clear ComplianceKit pitch
  • Awake-at-night staff rule (night-care facilities) — unique NV requirement; operational complexity
  • Accommodation Facility category (casino / hotel drop-in) — hyper-local Vegas niche; episodic staffing → outsized compliance cost
  • Nevada Registry enrollment required — ComplianceKit integration surfaces here
  • Substantiated complaint in Aithent — 12-mo buyer window post-finding
  • Wildfire / AQI outdoor-play guidance — climate-specific emergency-ops module fit
  • Post-consolidation data gaps — old URLs 404, contact info in flux; ComplianceKit's "stay current" value prop lands especially well here
