6. Violations & Inspection Data

Primary sources

  • OCFS Child Care Facility Search: ocfs.ny.gov/programs/childcare/looking/ — per-facility Program Profile pages expose violation history for the prior 2 years.
  • OCFS Enforcement Action list: ccfs-fie.php — facilities currently under probation, pending revocation, suspension, or denial.
  • NYC DOHMH Child Care Connect: a816-healthpsi.nyc.gov — NYC-regulated centers only.

Data format

State cb42-qumz dataset contains 33 columns but no structured violation field — only a Facility Status (Pending Revocation, Suspended, etc.) and a URL to each facility's HTML Program Profile where violations are rendered. OCFS violation severity tiers: Class I (serious/imminent), Class II (public health; max fine $450/day first offense), Class III (minor; max $400/day first offense).

NYC dsg6-ifza publishes structured violation rows: inspection_date, inspection_status, violation_category, violation_rate_per_child, health_violation_rate_per_child, regulation_summary. Refresh: daily.

Freshness

  • cb42-qumz: weekly (facility status)
  • Program Profile HTML: 1–3 week lag after inspection close
  • NYC dsg6-ifza: daily

Scraping / access strategy

  • State: SODA API https://data.ny.gov/resource/cb42-qumz.json?$limit=50000; then for each row fetch the Program Profile URL and parse the violations HTML block. ASP.NET Core backend, no ViewState encryption on public route.
  • NYC: pure SODA query; filter by inspection_date and violation_category is not null.
  • Enforcement: scrape the static ccfs-fie.php page for a hot-leads list.

FOIA path

NY FOIL (Public Officers Law Art. 6) — OCFS FOIL. Used for pre-2-year violation history and un-redacted investigation narratives.

7. Outreach Signals

  • Facility Status = "Pending Revocation" / "Suspended" (available via cb42-qumz SODA query) — highest-intent signal.
  • New Class II/III violation in last 30 days (requires Program Profile HTML scrape).
  • License expiring in next 90 days (available — license_expiration_date column).
  • Appears on Enforcement Action page (available via simple static scrape).
  • NYC: SRNC-equivalent Critical Violation in last 90 days (available via dsg6-ifza SODA query).
