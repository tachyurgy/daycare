7Violations & Inspection Data

Access mode

Per-facility lookup   No bulk export

MSDH publishes inspection history and cited violations on the per-facility page inside the Child Care Facility Search. There is no statewide violations CSV.

Class I penalty schedule

Monetary penalties assessed automatically on significant findings:

  • First Class I occurrence: $500
  • Subsequent Class I occurrences: $1,000

Key fields exposed per facility

  • Facility name, license number, license status
  • Inspection date & type (routine / complaint / follow-up / renewal)
  • Violation rule section (mapped to Subpart 55 Chapter 1 & 2)
  • Severity class + corrective action + resolution date
  • Monetary penalty amount (when assessed)

Scraping strategy

  1. Enumerate facility list from existing mississippi_leads.csv (1,414 rows).
  2. Playwright or requests-with-ViewState session against ccsearch.aspx.
  3. Parse per-facility detail HTML table; normalize violation codes to Subpart 55 section IDs.
  4. Optional: cross-key against MARIS ID field for lat/lon to build violation heatmap.

FOIA path

Mississippi Public Records Act (Miss. Code §25-61-1 et seq.). MSDH is the custodian for child care violation records. Request via the MSDH public records request form, citing Miss. Code §43-20-8 and asking for "all inspection reports and violation histories for [license #s] in [date range]." Typical statutory response: 7 working days.

Violations / inspection data sources

MSDH Child Care Facility Regulatory Violations page MSDH Child Care Facility Search (per-facility inspection history) MSDH Complaints page (public records request instructions) Alternate Facility Search URL

  • Strong phone coverage — ~95% phone fill, normalized (XXX) XXX-XXXX. Ready for outbound dialer.

  • Class I penalties start at $500 — a provider's first citation is a high-intent sales moment. Scrape the per-facility violation page weekly, identify new Class I assessments, trigger personalized outreach.
  • MDHS subsidy-receiving providers (via ECCD find-a-provider) are a subsegment more tolerant of compliance software subscription costs.
