What this atlas contains

For every U.S. state, four artifacts:

  • compliance.md — licensing authority, primary reg cite, facility types, forms, ratios, records, inspection + renewal cadence, state-specific flags.
  • SOURCES.md — exact URL, format, row count, extraction method, and limits for the provider contact list ({state}_leads.csv at project root).
  • violations.md — where recent violations/deficiencies/failed inspections are published, format, freshness, scraping/access strategy, FOIA fallback.
  • data_report.html — self-contained stakeholder-ready report linking every source. One per state, listed below.
No fabrication. Every contact row was pulled from authoritative public sources. States where bulk data was unavailable document the authoritative search portal and use childcarecenter.us as a fallback for major-city samples (marked Bronze below). Every limitation is disclosed in the per-state SOURCES.md.

Violation-data access patterns across the country

Only two states expose a true Socrata/JSON API for per-facility inspection data: Texas and the Municipality of Anchorage (inside Alaska). The rest fall into four access tiers.

Tier 1 — Bulk API / Socrata / ArcGIS (minimal engineering)

  • Texas — HHSC publishes every deficiency within 14 days + data.texas.gov CCL operations dataset.
  • Delaware — data.delaware.gov exposes wb83-pkcv (~12.4k compliance review rows) + pnbd-85r6 (~1.9k complaint reviews), joinable on license_number.
  • Montana — SansWrite JSON API reverse-engineered; full state coverage.
  • Georgia — DECAL Enforcement Actions page has a CSV export.
  • Illinois — DCFS Sunshine portal has a CSV export button.
  • Nebraska — monthly negative-action PDFs at predictable URL pattern {MM}-{YY}ccnegactions.pdf.
  • Idaho — idahochildcarecheck.org/provider/{id} HTML, no auth/captcha.
  • Wisconsin — Angular JSON API + daily-refresh XLSX.
  • Washington — DCYF Apex Remoting JSON endpoint.
  • New Jersey — NJCCIS JSON backing API for per-provider detail pages.
  • Anchorage, AK — Socrata abmr-hyh6 (2008-present, full inspection dataset).

Tier 2 — Deterministic URL scrape (ASP.NET/legacy CGI)

  • Ohio — deterministic PDF URL pattern /pdf/{programId}_{YYYY-MM-DD}_{TYPE}.pdf.
  • Virginia — legacy dss.virginia.gov/facility/search/cc2.cgi?ID={fac}&Inspection={insp}.
  • West Virginia — classic ASP get_details.asp?q={provider_number}.
  • Utah — ccl.utah.gov/ccl/public/checklist/{ID}.
  • Louisiana — louisianaschools.com/schools/{FACILITY_CODE}/Inspections.
  • Oklahoma — sequential license-ID sweep on OKDHS Locator.
  • Maine — ASP.NET WebForms with ViewState on search.childcarechoices.me.
  • Iowa — Titan ComplianceReport ASP.NET.
  • Connecticut — eLicense ASP.NET with PDF DocumentIdnt attachments.
  • Indiana — FSSA Carefinder ASP.NET with viewstate.
  • Maryland — CheckCCMD dynamic PDF + ProviderInformation.aspx?ProvNum=.
  • South Carolina — scchildcare.org/provider/{id}/{slug}/.
  • Oregon — Find Child Care OR per-provider + CCLD-0093 Contact Report PDFs.
  • Minnesota — DHS Licensing Information Lookup per-facility.
  • Mississippi — MSDH provider search.
  • New York — OCFS CCFS profile pages + NYC DOHMH Socrata.
  • Pennsylvania — COMPASS provider search + DHS Quarterly Licensing PDFs.
  • Wyoming — Find Child Care WY (2019+ substantiated complaints only).
  • Tennessee — ServiceNow CSP + SWORPS.
  • New Mexico — ECECD inspection survey PDFs (3-yr window).

Tier 3 — Salesforce / modern SPA (Playwright required)

  • Arizona — AZ Care Check Lightning (daily refresh, primary-source verified).
  • Michigan — CCHIRP Salesforce + 5-yr rolling disciplinary HTML list.
  • Colorado — Colorado Shines detail pages.
  • Massachusetts — EEC LEAD Salesforce Exp. Cloud.
  • Kentucky — kynect Provider Details Salesforce Lightning.
  • New Hampshire — NH Child Care Search Salesforce.
  • Rhode Island — RISES Salesforce LWC.
  • Arkansas — DCCECE eLicensing Salesforce (returns 403 to non-JS).
  • Vermont — BFIS session-protected app.
  • North Dakota — ND DHHS CCL System (2023 launch).
  • South Dakota — SD OLA Constituent Portal.
  • Nevada — Aithent ALiS portal (post-2024 reorg).
  • Kansas — KDHE OIDS (captcha-locked).
  • North Carolina — DCDEE Telerik-based search.

Tier 4 — Records request only

  • Hawaii — DHS BESSD has not posted serious-incident data online since 2016; UIPA request (HRS Ch. 92F) is the only path.
  • Alabama — deficiency reports posted at the facility, not online; AL Open Records Act 30-60 day response.
  • Missouri — DHSS legacy app blocks POST; Sunshine Law request is the fastest path.
  • Florida — CARES is per-facility only; FL Sunshine Law (F.S. Ch. 119) is unusually liberal and is the strongest records-request lever in the country.
  • California — CCLD Facility Search has PDFs April 2015+; LIS bulk = CPRA (months-long).

Strategy: finding daycares with recent violations or failed inspections

The GTM play is an automated "recently cited" signal that feeds sales triggers. Every state has a path; the engineering cost varies from a nightly cron job (Tier 1) to a quarterly records request (Tier 4).

Recommended build order (by ROI)

  1. Phase 1 — API states (weeks 1-3). Build connectors for TX, DE, MT, GA, IL, WI, WA, NJ, NE, ID + Anchorage, AK. These yield the richest structured violation data for the lowest engineering cost. Daily cron pulling any facility with a new violation logged in the prior 7 days feeds the CRM immediately.
  2. Phase 2 — Deterministic URL scrapes (weeks 4-8). OH, VA, WV, UT, LA, OK, ME, IA, CT, IN, MD, SC, OR, MN, MS, NY, PA, WY, TN, NM. Same cadence + daily, but each state is ~200 lines of Python/Playwright. Adds ~20 states of coverage.
  3. Phase 3 — Salesforce SPAs (weeks 8-14). AZ, MI, CO, MA, KY, NH, RI, AR, VT, ND, SD, NV, KS, NC. Each needs Playwright with session cookies, CAPTCHA/rate-limit handling. Run weekly not daily.
  4. Phase 4 — FOIA automation (ongoing). HI, AL, MO, FL, CA. Template requests per state, track response SLA (AR 3 business days, FL rolling, AL 30-60 days, HI weeks). FL's Sunshine Law is the single highest-leverage records regime in the country; CA's CPRA is worth the wait for LIS extract.

Signals worth tracking on each facility

  • Recent violation — any non-compliance finding in last 30/60/90 days. Highest-intent signal.
  • Repeat violation — same regulation cited twice in 12 months. Indicates systemic issue; high conversion.
  • Enforcement action — probationary status, consent decree, civil penalty, license suspension/revocation pending. Highest urgency.
  • Failed reinspection — CAP not accepted. Indicates operator is struggling with paperwork — ICP for ComplianceKit.
  • New license — first 90 days is onboarding-pain window; convert before they buy a competitor.
  • Renewal-due — 90/60/30 days to license expiration. Leverage-with-urgency.
  • Complaint investigation pending — public-record signal (some states publish) that operator is under pressure.
  • Star-rating drop — for QRIS states (TN, KY, GA, OK, WI, CO, TX Rising Star, etc.) a rating downgrade precedes a subsidy hit.

Journalism-derived "catalyst markets"

  • Rhode Island — Aug 2025 Auditor General audit found H&S violations at all 50 audited facilities, 60% in significant non-compliance. DHS has pledged to visit every provider. Urgency is real.
  • Maine — Maine Monitor + Bangor Daily News: 16,000+ citations since 2021, 1,481 facilities, zero revocations, zero fines. Operator reputation pressure is mounting.
  • Massachusetts — 2024 state audit + Boston 25 "25 Investigates" series documented systemic oversight gaps.
  • Vermont — March 2026 State Auditor report flagged serious vs. standard classification inconsistencies. Active policy lens.
  • New Mexico — Universal Child Care rollout (Oct 2025) + new Wonderschool portal (Feb 2026) = immediate paperwork burden.
  • Kansas — KDHE → Office of Early Childhood transition July 1 2026 = URL churn + regulatory confusion.

Known limits & next steps

  • Email/website enrichment — most Gold-tier states publish phones but not emails. Run Hunter/Apollo/Clearout against business_name + phone + state to fill ~80% of the email column across the 120k rows.
  • Bronze-tier scale-up — 22 states have portal-only data and the provided CSVs are samples from major cities. Upgrade to full coverage requires Playwright harness (Tier 3 list) or records request (Tier 4).
  • CA LIS extract — file CPRA request now; typical response 3-6 months. Would yield every CA facility's inspection history in one shipment.
  • FL bulk download — test myflorida.com Sunshine Law channel; the state's liberal interpretation has previously yielded full databases.
  • NC Telerik automation — Playwright scripted form submission is the fastest path to 5,000+ NC facilities with full data (current CSV only has 1,152 from aggregator).
  • Data freshness — everything stamped 2026-04-18. Rebuild quarterly for provider CSVs; daily for Tier 1 violation feeds.
