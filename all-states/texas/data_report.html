6. Violations & Inspection Data

Texas is the gold standard for public child-care inspection data in the United States. Every deficiency at every licensed operation is posted within 14 days of the visit with full Minimum Standard citation, narrative, weight, corrective action, and resolution. Bulk access is available via the Socrata-backed Texas Open Data Portal.

Search Texas Child Care

Per-operation HTML with inline deficiency table. Deep-linkable operation pages. No scraping gymnastics required.

Texas Open Data Portal

Socrata-backed datasets. SODA REST API. CSV / JSON / XML / RDF / RSS export. Production-grade ingestion path.

Deficiency weight system

High / Medium-High / Medium / Medium-Low / Low — drives administrative penalty amount and enforcement escalation (evaluation → probation → suspension → revocation).

Records-request path

Texas Public Information Act (Gov. Code Ch. 552). 10-business-day initial determination. Useful for internal data dictionaries / narratives not already in Socrata.

7. Outreach Signals

  • Public deficiency postings = public reputation pressure — every operator's compliance history is google-able by parents. Reputation-management value prop resonates strongly.
  • Ratio-violation pain: 640 cited ratio violations/year in TX (Medium-High weight) — real-time ratio monitor is a direct hit.
  • Daily Building & Grounds Checklist (Form 1100) — 24 items, Monday-Friday — the single most operationally intensive recurring compliance task; digitizing it saves hours/week per director.
  • Plan of Operation (Form 2948) — operational blueprint, updated annually; maps to Policy Library module.
  • 24+24 training structure — 24 hrs pre-service (8 before child contact) + 24 hrs annual in-service. Staff training tracker is a natural Staff File sub-product.
  • Administrative penalty threat — 30-day hearing window creates urgency; providers with active citations have the highest buying intent.
  • Five MSA dominance: Houston, DFW, San Antonio, Austin, El Paso cover majority of centers. Geographic focus = efficient GTM.
  • Chapter 745 Subchapter F background-check rigor — ongoing re-check schedule = ongoing credential-tracking pain.

8. Next-Step Data Acquisition

  1. Pull HHSC CCL operations data from Socrata (`bc5r-88dy` and related sub-datasets). Full operations roster + deficiency table joined — the foundation of the Texas compliance knowledge graph.
  2. Cross-reference CACFP site-level datasets (`y9zm-iygj` centers, `ub2y-4i6h` homes, `x9ve-x3iy` meal reimbursement) for federal-participation flag — CACFP operators have higher paperwork tolerance and higher willingness to pay.
  3. Enrich `texas_leads.csv` with Socrata-derived license number, operation type, current deficiency count, current enforcement status, ages served, capacity.
  4. Scrape per-operation pages on childcare.hhs.texas.gov where Socrata fields lack narrative detail (typically newer visits).
  5. File TPIA request to HHSC for data dictionaries, Minimum Standard weight tables, and any enforcement decision memos used by CCR regional supervisors.
  6. Monthly Child Care Services Data Report (`c7ep-cuy6`) for workload / churn sizing; county-level (`xge9-adrm`) for geographic planning.
