2. Primary Regulatory Citation

Statute: SDCL Title 26, Chapter 26-6 (Child Welfare; Foster Homes, Shelters, and Child Care).

Administrative Rules: ARSD Chapter 67:42 (Child Care Services) — key articles: 67:42:10 (Registered Family Day Care), 67:42:15 (Before/After School), 67:42:17 (Licensed Centers & School-Age).

Background-check statute: SDCL 26-6-14.4 (state + FBI fingerprint + Central Registry for all adults with unsupervised access).

5. Licensing Forms

All application / renewal / incident-reporting / correction-plan activity routes through the OLA Constituent Portal. Published paper forms include:

  • Child Care Licensing / Registration Application (OLA portal)
  • DCI fingerprint card — criminal background check request
  • Central Registry check (Child Abuse & Neglect)
  • Medical / Health Statement (staff and enrolled children)
  • Immunization Record (SD DOH-approved)
  • Fire Marshal inspection report (centers)
  • Correction plan / variance request (online submission)

6. Violations & Inspection Data

Source

OLA Constituent Portal publishes 4 years of inspection history, compliance plans, corrective-action plans, and stipulation / consent agreements per provider profile.

Format

Per-facility HTML + PDF attachments. No bulk export JS-rendered

Access strategy

Headless browser (Playwright) to enumerate by zip/county; parse each provider profile; OCR the attached inspection PDFs.

Records-request path

SDCL 1-27 open-records request to DSS / OLA for machine-readable extract of preceding 7 years.

Full analysis: violations.md

7. Outreach Signals

  • Annual renewal cadence — every license/registration renews every 12 months. Renewal stress is a predictable, recurring buying trigger.
  • Public posting of violations — every SD provider has 3-4 years of inspection results visible to parents. Reputational stake in "clean" compliance is high.
  • Tight market (~700-750 programs) — full-TAM outreach is feasible in two weeks of sequencing.
  • Sioux Falls + Rapid City account for majority of licensed centers — concentrate first-wave outreach there.
  • CACFP / Head Start double-enrolled programs carry additional federal paperwork — higher pain tier, higher willingness to pay.
  • Background-check rigor (SDCL 26-6-14.4) — strict FBI + Central Registry + sex-offender triangulation creates ongoing credential-tracking need that maps cleanly to the Staff File module.

8. Next-Step Data Acquisition

  1. Scrape OLA portal via Playwright: enumerate all provider profiles, extract license-number + address + email (when shown) + phone + inspection-history URL set. Target: full statewide registry in ≤48 hours of compute.
  2. Download + OCR inspection PDFs for every profile. Normalize to (provider_id, date, type, rule_cited, corrected_by, narrative). Run ML classifier to tag severity (high/med/low) since SD doesn't publish a formal weight system.
  3. File SDCL 1-27 open-records request to DSS / OLA for: full inspection-database extract preceding 7 years, all enforcement actions, all stipulation / consent agreements. Targeted fields per violations.md.
